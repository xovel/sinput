# sinput

A search input plugin, based on jQuery.

This is branch of `2.x`. For `1.x`, visit [sinput 1.x](https://github.com/xovel/sinput/tree/1.x)

## Build

1. `git clone git@github.com:xovel/sinput.git`
2. `npm install`
3. `gulp` or `gulp build`
